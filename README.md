# fattyarrow
![screencast](./tty.gif)
A zsh theme

## Features
- time (hh:mm:ss)
- git support
- virtualenv support
- nvm support
- last command's exit code

## How-to-install
1. Run this command.
```
curl https://raw.githubusercontent.com/sohnryang/fattyarrow/master/fattyarrow.zsh-theme > $ZSH_CUSTOM/themes/fattyarrow.zsh-theme
```
2. Set your `$ZSH_THEME` to `fattyarrow`.
3. Enjoy!
